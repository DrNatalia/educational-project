# Коротко о пройденном на курсе _ЯП_.

### Делаем папку **репозиторием**

``` 
git init

```

---

### Проверяем состояние репозитория

``` bash
git add -all

```

или

``` bash
git add

```

---

### Делаем первый **коммит**

``` git commit -m 'Текст изменений'

```

### Отправляем изменения на удаленный репозиторий

```git push

```
